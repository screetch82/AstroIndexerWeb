2. Übersicht der Datenverarbeitung

Diese Website erhebt und verarbeitet so wenig personenbezogene Daten wie möglich. Es werden keine Cookies gesetzt, kein Tracking eingesetzt und keine Analyse-Dienste (z. B. Google Analytics) verwendet. Es werden keine Daten an Werbenetzwerke oder Social-Media-Plattformen übermittelt.

3. Hosting

Diese Website wird über GitHub Pages (GitHub, Inc., 88 Colin P Kelly Jr St, San Francisco, CA 94107, USA) bereitgestellt. Beim Abruf der Website werden durch GitHub automatisch Server-Logdaten erhoben, darunter:

  • IP-Adresse des zugreifenden Geräts
  • Datum und Uhrzeit des Zugriffs
  • Aufgerufene Seite (URL)
  • HTTP-Statuscode
  • Übertragene Datenmenge
  • Browser-Typ und Betriebssystem (User-Agent)

Die Verarbeitung erfolgt auf Grundlage von Art. 6 Abs. 1 lit. f DSGVO (berechtigtes Interesse an der sicheren und effizienten Bereitstellung der Website). Die Server-Logdaten werden von GitHub verarbeitet; nähere Informationen finden Sie in der GitHub Privacy Statement.

GitHub, Inc. unterliegt als US-Unternehmen dem EU-U.S. Data Privacy Framework (DPF). Die Zertifizierung können Sie unter dataprivacyframework.gov einsehen.

4. Software-Downloads

Die Software-Installationsdateien werden über Backblaze B2 (Backblaze, Inc., San Mateo, CA, USA) bereitgestellt. Beim Download werden durch Backblaze die für die Auslieferung technisch notwendigen Daten verarbeitet (IP-Adresse, Zeitpunkt, angeforderter Dateiname).

Rechtsgrundlage: Art. 6 Abs. 1 lit. b DSGVO (Vertragserfüllung bzw. vorvertragliche Maßnahme — Bereitstellung der angeforderten Software).

Nähere Informationen: Backblaze Privacy Policy.

5. Testlizenz-Download

Auf der Website steht eine Testlizenz-Datei (JSON) zum direkten Download bereit. Dieser Download erfordert keine Eingabe personenbezogener Daten (kein Name, keine E-Mail-Adresse, kein Konto). Die Datei wird direkt von GitHub Pages ausgeliefert; es gelten die unter Abschnitt 3 genannten Bedingungen.

6. Lokale Speicherung (localStorage)

Diese Website speichert Ihre Sprachauswahl im localStorage Ihres Browsers unter dem Schlüssel astroindexer_language. Dabei handelt es sich nicht um ein Cookie, sondern um eine rein lokale Speicherung auf Ihrem Gerät. Der gespeicherte Wert (z. B. „de“, „en“, „es“) wird nicht an den Server übermittelt und dient ausschließlich der Komfortfunktion, Ihre Spracheinstellung beim nächsten Besuch wiederherzustellen.

Rechtsgrundlage: Art. 6 Abs. 1 lit. f DSGVO (berechtigtes Interesse an einer nutzerfreundlichen Website). Sie können den localStorage-Eintrag jederzeit über die Entwicklertools Ihres Browsers löschen.

7. Kontaktaufnahme per E-Mail

Wenn Sie uns per E-Mail kontaktieren, werden Ihre Angaben (Name, E-Mail-Adresse, Inhalt der Nachricht) zur Bearbeitung Ihrer Anfrage verarbeitet und gespeichert. Eine Weitergabe an Dritte erfolgt nicht.

Rechtsgrundlage: Art. 6 Abs. 1 lit. b DSGVO (vorvertragliche Maßnahme oder Vertragserfüllung) bzw. Art. 6 Abs. 1 lit. f DSGVO (berechtigtes Interesse an der Beantwortung von Anfragen).

Ihre Daten werden gelöscht, sobald die Anfrage abschließend bearbeitet ist und keine gesetzlichen Aufbewahrungspflichten entgegenstehen.

8. Keine weiteren Dienste

Diese Website verwendet keine:

  • Cookies (weder technische noch Tracking-Cookies)
  • Web-Analytics oder Tracking-Dienste
  • Externen Schriftarten (z. B. Google Fonts)
  • Content-Delivery-Networks (CDN) für CSS oder JavaScript
  • Social-Media-Plugins oder Embeds
  • Werbenetzwerke oder Retargeting

Alle Stylesheets und Skripte werden direkt von GitHub Pages ausgeliefert.
